Detailed Description

The Mstch module.

Mustache template handling using {{ bustache }}. Boost needs to be installed in your project to use bustache.

Example Usage: 

Mstch mstch( "{{mustache}} templating", bustache::object{{"mustache", "bustache"}} );
## result "bustache templating" ##
